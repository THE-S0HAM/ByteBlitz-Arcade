ByteBlitz Arcade Assets

This directory contains global assets used by the arcade hub:

/fonts - Arcade-style fonts for text display
/images - Icons, backgrounds, and UI elements
/sounds - Sound effects and music

Recommended fonts:
- arcade.ttf - Classic arcade font
- pixel.ttf - Pixelated retro font

You can download free arcade-style fonts from sources like:
- https://www.dafont.com/arcade-classic.font
- https://www.1001fonts.com/pixel+video-game-fonts.html
